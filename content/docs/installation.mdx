# DELIGENIUS

> Deligenius is a lightweight, high-performance and customizable web framework for [Deno](https://deno.land/).
> It adapts from experss.js and koa.js and build using only [Typescript](https://www.typescriptlang.org/),
> which provides strong typing experience and makes your server more robust.

### Installation

Please make sure you have [Deno runtime](https://deno.land/) installed. You can also follow the
[Deno official installation guide](https://deno.land/#installation) or using the installers below:

**Shell (Mac, Linux):**

```bash
curl -fsSL https://deno.land/x/install/install.sh | sh
```

**PowerShell (Windows):**

```powershell
iwr https://deno.land/x/install/install.ps1 -useb | iex
```

**Homebrew (Mac):**

```bash
brew install deno
```

---
### Check your installation
Make sure the installation is valid, use the command below.

```bash
$ deno --version
```

if you get something like this, you are good to go.

```bash
deno 1.0.3
v8 8.4.300
typescript 3.9.2
```
