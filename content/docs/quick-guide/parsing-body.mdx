### Use BodyParser

> Deligenius has an official middleware module: `bodyParser`,
> `mid` is a shortcut of [bodyParser](https://github.com/deligenius/bodyparser)
> Please make sure you have the `mid` import

```ts
import { mid } from "https://raw.githubusercontent.com/deligenius/bodyparser/master/mod.ts"
```

#### Get `json` data

```ts
app.use(mid.json(), ctx => {
  let json = ctx.req.json
  ctx.send(json)
})
```

#### Get `xml` data

```ts
app.use(mid.xml(), ctx => {
  let xml = ctx.req.xml
  ctx.send(xml)
})
```

#### Get `grqphql` data

```ts
app.use(mid.graphql(), ctx => {
  let graphql = ctx.req.graphql
  ctx.send(graphql)
})
```

#### Get `text` data

```ts
app.use(mid.text(), ctx => {
  let text = ctx.req.text
  ctx.send(text)
})
```
