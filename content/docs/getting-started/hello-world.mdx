### Hello world example

The server starts and listens on `port:8000` for requests.
It responsed every request with "Hello World".

```ts
// ./app.ts
import { Application } from "https://deno.land/x/deligenius/mod.ts"

let app = new Application({ port: 8000 }).listen()

app.use(ctx => {
  ctx.send("Hello World!")
})
```


### Run server locally

Create a directory named `server`. Then create a file named
`app.ts` then copy the code above. Your directory should be like this:

```
server/
└── app.ts
```

Run the app with command below:

```bash
> deno run --allow-net ./app.ts
```
